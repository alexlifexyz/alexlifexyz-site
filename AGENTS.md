# Alex Personal Site

This file is the default session bootstrap for this repo.

If you start a new session in this directory, treat the rules below as already loaded.

## What This Repo Is

This is the codebase for Alex's personal website.

The site is expected to combine:

- `个人博客`
- `项目展示`
- `公开简历`
- `持续更新的个人品牌站`

It is not a generic dev portfolio.
It is not a random template demo.
It is not an AI news mirror.

## Core Identity

Always preserve this identity:

`一个 10 年 Java 后端，经历过大厂裁员，现在在转 AI 应用和独立开发。`

The site should feel like a real engineer in transition, not a polished fake founder page.

## Product Priorities

When making decisions, use this order:

1. clarity of identity
2. readability of articles
3. credibility for interview / collaboration
4. maintainability across multiple devices and sessions
5. aesthetics

## Non-Negotiable Rules

- Do not bury the identity under generic AI language.
- Do not make the homepage just a reverse-chronological blog list.
- Do not over-design the first version.
- Prefer simple static architecture over premature dynamic systems.
- Public content should come from curated, publish-ready material only.
- Internal working docs, drafts, metrics, and raw sources should stay outside this repo unless intentionally exposed.

## Context Read Order

1. `docs/PROJECT-BRIEF.md`

3. `docs/ROADMAP.md`
4. `docs/SESSION-HANDOFF.md`
5. `docs/DECISIONS.md`
6. `docs/DEVELOPMENT-WORKFLOW.md`
7. `README.md`

## Collaboration Rules

- Any meaningful product decision should be recorded in `docs/DECISIONS.md`.
- Any unfinished work or next actions should be recorded in `docs/SESSION-HANDOFF.md`.
- Any scope changes should update `docs/MVP-SCOPE.md` or `docs/ROADMAP.md`.
- Keep changes easy to understand for a future session on another machine.

## Design Direction

The visual language should be:

- calm
- sharp
- engineering-first
- readable
- not overly corporate

Avoid:

- SaaS landing page clichés
- oversized hype copy
- purple-on-white generic startup aesthetics
- fancy motion before content structure is solid
