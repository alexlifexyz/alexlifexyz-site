# MVP Scope

## MVP Goal

Ship a first public version quickly.

The first version should be simple, credible, readable, and easy to extend.

## MVP Pages

- `Home`
- `Writing`
- `Post detail`
- `Projects`
- `About`
- `Resume`
- `Now`

## MVP Features

- responsive layout
- basic SEO metadata
- article list
- article detail pages
- project list
- markdown-based content
- custom domain readiness
- analytics-ready script slot

## MVP Content

Initial content target:

- `5-8` published articles
- `2-3` project pages
- `1` resume page
- `1` about page
- `1` now page

## Out Of Scope For MVP

- comments
- full-text search
- CMS
- auth
- newsletter system
- complex animations
- tag system beyond the basics
- multilingual support
