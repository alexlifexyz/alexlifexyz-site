# Roadmap

## Phase 0: Foundation

- define site identity
- define information architecture
- define content source boundaries
- create collaboration rules

## Phase 1: MVP

- initialize Astro site
- build homepage
- build writing list and post pages
- build project pages
- build about / resume / now pages
- import first batch of public articles
- deploy to Cloudflare Pages
- connect custom domain

Current status:

- MVP is deployed to `https://alexlifexyz-site.pages.dev/`
- visual polish is still pending
- content import is still pending
- custom domain is still pending

## Phase 2: Content Operations

- define article frontmatter schema
- improve series pages
- improve article cards and metadata
- add featured posts
- add basic post filters

## Phase 3: Credibility Layer

- refine resume page for interview use
- add timeline / experience blocks
- improve project case study structure
- add selected writing highlights to homepage

## Phase 4: Growth Layer

- add lightweight analytics dashboard notes
- add lead capture or contact funnel
- add better internal linking
- add selected landing pages for specific topics

## Phase 5: Product Layer

- evaluate small interactive demos
- evaluate AI app showcase pages
- decide whether dynamic features justify upgrading architecture
