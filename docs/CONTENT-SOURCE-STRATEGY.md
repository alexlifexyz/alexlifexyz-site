# Content Source Strategy

## Public vs Internal Content

This site should only contain public-ready material.

## Public-Ready Content

Suitable for this repo:

- published articles
- polished evergreen essays
- project case studies
- public bio / resume content
- now page updates

## Internal-Only Content

Should stay in the WeChat content workspace repo:

- raw source captures
- experimental drafts
- analytics records
- publishing packages
- internal prompts
- operation notes

## Import Strategy

Phase 1:

- manual curation
- copy selected public content into site content collections

Phase 2:

- add a lightweight sync workflow if the public source format becomes stable
